How to Display Full Width Images in Hugo

Hongtao Hao / 2020-11-02

I originally thought it must be super difficult to have full width images in Hugo because the width of my body text is already defined and yet all figures have to be within body. I stumbled upon a very easy solution here .

Add the following1 to your website stylesheet:

.fullwidth {
  width: 100vw;
  position: relative;
  left: 50%;
  right: 50%;
  margin-left: -50vw;
  margin-right: -50vw;
}

Then, when using Hugo’s default figure shortcode, include class="fullwidth":
