Hugo 如何代码高亮

郝鸿涛 / 2020-01-21

Hugo 目前自带代码高亮功能，官网的说明 列出如下的设置选项 （以 toml 格式为例）：

[markup]
  [markup.highlight]
    codeFences = true
    guessSyntax = false
    hl_Lines = ""
    lineNoStart = 1
    lineNos = false
    lineNumbersInTable = true
    noClasses = true
    style = "monokai"
    tabWidth = 4

大家可以试一下具体每个设置的功能，我这里主要讲一下 style 的设置，因为这个不是一个布尔值，而是需要输入具体内容。

Hugo 的代码高亮功能是由 chroma 提供的。
